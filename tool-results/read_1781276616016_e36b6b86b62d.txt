   344→     - `comment-added` — Broadcasts `new-comment` to all in room
   345→     - `comment-resolved` — Broadcasts `comment-update` to all in room
   346→     - `element-selected` — Broadcasts `selection-update` to other users
   347→     - `chat-message` — Broadcasts `new-chat-message` to all in room
   348→     - `disconnect` — Removes user from room, broadcasts `user-left`, cleans up empty rooms
   349→   - **State tracked per room:** users Map (socketId → CollabUser), designTree
   350→   - **Cursor colors:** 10-color rotating palette (rose, violet, amber, cyan, pink, lime, orange, indigo, teal, rose-600)

   355→   - Connects via `io('/?XTransformPort=3003')` — NEVER uses localhost/port in URL

   361→     - `cursors` — Map of userId → CursorPosition (derived, filtered to current room users)

   363→     - `emitCursorMove(x, y)` — Send cursor position
   364→     - `emitElementSelected(elementId)` — Send element selection

   372→     - `onNewChatMessage(callback)` — Register listener for chat messages
