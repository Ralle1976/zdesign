   336→   336→2. **`/mini-services/collab-service/index.ts`** — Socket.io server on port 3003
   337→   337→   - HTTP server with socket.io, path `/`, CORS enabled for all origins
   338→   338→   - Room-based architecture keyed by projectId
   339→   339→   - **Events handled:**
   340→   340→     - `join-project` — User joins a project room; broadcasts `user-joined`, sends `project-state` (user list + design tree) to joiner
   341→   341→     - `leave-project` — User leaves room; broadcasts `user-left`
   342→   342→     - `cursor-move` — Broadcasts `cursor-update` to other users in room
   343→   343→     - `design-update` — Stores latest design tree in room state, broadcasts `design-changed` to others
   344→   344→     - `comment-added` — Broadcasts `new-comment` to all in room
   345→   345→     - `comment-resolved` — Broadcasts `comment-update` to all in room
   346→   346→     - `element-selected` — Broadcasts `selection-update` to other users
   347→   347→     - `chat-message` — Broadcasts `new-chat-message` to all in room
   348→   348→     - `disconnect` — Removes user from room, broadcasts `user-left`, cleans up empty rooms
   349→   349→   - **State tracked per room:** users Map (socketId → CollabUser), designTree
   350→   350→   - **Cursor colors:** 10-color rotating palette (rose, violet, amber, cyan, pink, lime, orange, indigo, teal, rose-600)
   351→   351→   - Graceful shutdown on SIGTERM/SIGINT with socket cleanup
   352→   352→
   353→   353→3. **`/src/hooks/useCollaboration.ts`** — Frontend collaboration React hook
   354→   354→   - `useCollaboration(projectId, userId, userName)` hook
   355→   355→   - Connects via `io('/?XTransformPort=3003')` — NEVER uses localhost/port in URL
   356→   356→   - Auto-joins project room when projectId changes
   357→   357→   - Re-joins on reconnection
   358→   358→   - **Returns:**
   359→   359→     - `isConnected` — Connection status
   360→   360→     - `users` — Array of CollabUser in current room
   361→   361→     - `cursors` — Map of userId → CursorPosition (derived, filtered to current room users)
   362→   362→     - `selections` — Map of userId → SelectionInfo (derived, filtered to current room users)
