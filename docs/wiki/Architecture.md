## File-Based Extensibility

```
skills/               вҶҗ Drop a SKILL.md = add a skill
design-systems/       вҶҗ Drop a DESIGN.md = add a system
panelists/            вҶҗ Drop a .md = add a panelist
audits/               вҶҗ Drop a .md = add an audit check
mcp/tools/            вҶҗ Drop a .ts = add an MCP tool
exporters/            вҶҗ Drop a .ts = add an export format
```

## Technology Stack

- **Framework**: Next.js 16 (App Router, Turbopack)
- **Runtime**: Bun
- **AI**: Z.ai GLM-5.2 (Anthropic-compatible endpoint, direct)
- **Database**: Prisma + SQLite
- **State**: Zustand
- **Styling**: Tailwind CSS v4 + shadcn/ui
- **MCP**: Streamable HTTP (localhost)
- **License**: Polyform Noncommercial 1.0.0
